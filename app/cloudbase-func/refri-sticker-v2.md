# RefriSticker Image Prompt

> version: v2
> 接入火山引擎 Ark Doubao Seedream 5.0 lite,采用多图融合:
> 图 1 = 用户上传的旅行照片（提供主体识别度与场景结构）
> 图 2 = 风格参考图（提供艺术语言）
> 组合方式: `base + style.<风格> + composition.<比例> + negative`
> Vite 开发服务器自动热更新该文件。

## base

```text
将图 1 的真实旅行场景，按图 2 的艺术语言重新绘制为一张完整的旅行纪念插画。

保留图 1 的拍摄视角、主体（建筑、人物、自然景物）、空间关系和时间氛围，
使熟悉原地点的人仍然能辨认这张图。
不要添加图 1 中没有的人物、建筑、车辆或地标。

输出单幅平面插画，不要生成实体冰箱贴样机、产品外壳、桌面背景或 3D 厚度。
不要在画面中写任何文字——文案、日期、边框和水印由网站后续叠加。
```

## style.illustration

```text
清新复古的旅行海报插画：清晰而简化的轮廓和色块，平涂质感，
控制在 6 至 8 种主要颜色，配色明亮但不过度饱和，加入细腻的纸张颗粒。
整体具有现代文创商品和收藏海报的精致感。
```

## style.chinese

```text
当代国风工笔旅行插画：细致而克制的墨线轮廓，雅致的矿物色与低饱和
青绿、赭石、米白配色，适量宣纸肌理和东方留白。保留真实地点与现代物体，
不要把现代旅行场景改造成古代建筑或历史人物场景。
整体像现代博物馆文创，而不是传统山水画复制品。
```

## style.comic

```text
明快精致的旅行漫画插画：干净有力的线稿，清晰的平涂色块，赛璐璐阴影，
适度夸张但保持地点、人物身份与空间结构可辨认。配色活泼清爽，
避免密集网点、对话框、拟声词和分镜边框，
整体像一张单幅收藏级旅行漫画海报。
```

## composition.portrait

```text
4:5 竖版构图，主体居中或偏上，适合人物、建筑或纵深街景。
```

## composition.square

```text
1:1 方形构图，主体集中且居中，适合合照、食物或近景。
```

## composition.landscape

```text
4:3 横版构图，突出横向空间与全景关系，适合风景全貌或多入合照。
```

## negative

```text
禁止：文字、字母、数字、Logo、水印、签名、印章、乱码、重复人物、
额外肢体、畸形人脸、凭空增加物体、改变地标结构、产品样机、桌面背景、
3D 厚度、强烈摄影滤镜。
```
