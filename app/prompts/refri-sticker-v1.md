# RefriSticker Image Prompt

> version: v1
> 使用说明：编辑本文件后，Vite 开发服务器会自动热更新。代码会按 `base + style.* + composition.* + negative` 的顺序组合提示词。

## base

```text
请将用户上传的旅行照片转换成一张高完成度的旅行纪念插画。

严格保留原照片中可辨认的拍摄视角、主体、建筑结构、道路关系、自然景物以及人物和车辆的数量与相对位置，使熟悉该地点的人仍然能够辨认原场景。保留原照片的时间、天气和整体情绪，不要添加照片中不存在的人物、建筑、车辆、飞鸟或标志性物体。

整张画面必须使用统一的艺术语言，明确呈现为人工绘制的旅行插画，避免只对照片应用滤镜。

只输出平面正视图。不要生成实体冰箱贴样机、产品外壳、桌面、背景环境、立体厚度或投影。不要生成任何中文、英文、字母、数字、Logo、签名、印章、水印或乱码。文案、日期、边框和水印由网站后续叠加。
```

## style.illustration

```text
风格采用清新复古的旅行海报插画：清晰而简化的轮廓和色块，平涂与丝网印刷质感，控制在 6 至 8 种主要颜色，配色明亮但不过度饱和，加入细腻的纸张颗粒。树木、天空、人物与建筑使用统一的编辑插画语言，具有现代文创商品和收藏海报的精致感。
```

## style.chinese

```text
风格采用当代国风工笔旅行插画：细致而克制的墨线轮廓，雅致的矿物色与低饱和青绿、赭石、米白配色，适量宣纸肌理和东方留白。保留真实地点和现代物体，不要把现代旅行场景改造成古代建筑或历史人物场景。整体像现代博物馆文创，而不是传统山水画复制品。
```

## style.comic

```text
风格采用明快精致的旅行漫画插画：干净有力的线稿，清晰的平涂色块，赛璐璐阴影，适度夸张但保持地点、人物身份与空间结构可辨认。配色活泼清爽，避免密集网点、对话框、拟声词和分镜边框，整体像一张单幅收藏级旅行漫画海报。
```

## composition.portrait

```text
输出完整的 4:5 竖版插画。主体构图适合人物、建筑或纵深街景，画面从顶部到底部都用于插画，不需要预留文案空白；网站会在插画下方额外增加独立文案区。
```

## composition.square

```text
输出完整的 1:1 方形插画。主体构图平衡、集中，适合合照、食物或近景，画面从顶部到底部都用于插画，不需要预留文案空白；网站会在插画下方额外增加独立文案区。
```

## composition.landscape

```text
输出完整的 4:3 横版插画。构图突出横向空间、风景全貌或多人关系，避免把主体挤在画面中央。画面从顶部到底部都用于插画，不需要预留文案空白；网站会在插画下方额外增加独立文案区。
```

## negative

```text
禁止：文字、字母、数字、Logo、水印、签名、印章、乱码、重复人物、额外肢体、畸形人脸、凭空增加物体、改变地标结构、产品样机、桌面背景、3D 厚度、强烈摄影滤镜。
```
